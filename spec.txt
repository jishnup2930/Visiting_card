OVERVIEW

This HR Management System is a command-line tool built in Python, 
designed to manage employee data and leave records in a database.
It offers functionalities such as importing employee data from CSV files,
querying employee information, managing leave status, and performing table operations like deletion, 
updates, and removals.


INPUT:

Csv_file: Each row in the csv_file should contains the  following columns (sample csv file in folder sample data)

        firstname, lastname, designation, email, phonenumber


Model of vcard         

      * This is a sample vcard file
        
        BEGIN:VCARD
        VERSION:2.1
        N:Bob;John
        FN:John Bob
        ORG:Authors, Inc.
        TITLE:Lawyer
        TEL;WORK;VOICE:666.808.0750x9935
        ADR;WORK:;;100 Flat Grape Dr.;Fresno;CA;95555;United States of America
        EMAIL;PREF;INTERNET:.johnbob@com
        REV:20150922T195243Z
        END:VCARD

        * The qr code contains this data



OBJECTIVE:

The handle_initdb function initialize a sql database

The function handle_load function load a csv file in to table

The function generate_one_vcard create one vcard
 
The function generate_one_qrcode generate one qr code and save into a subdirectory qr_code

The function handle_query fetch the data from database (-c --vacrd,-q --qrcode )

The function handle_leave to add leave into leaves table
 
The function handle_leave_count get the leave remaining of a employee
 
The function handle_delete delete a table

The function handle_update update the table 

The function handle_remove  remove row in a table


USAGE

 - initdb: Initialize the database.
 - load: Import employee data into the database from a CSV file.
 - query: Retrieve information for a single employee.
 - leave: Update leave status for an employee.
 - leave_count: Check the remaining leave count for an employee.
 - delete: Delete a specific table.
 - update: Edit a table (update specific entries).
 - remove: Remove a row from a table


EXAMPLE
      python hr.py initdb

      python hr.py load employees.csv

      python hr.py query 123 -c -q
      
      python hr.py leave 123 '2023-11-22' 'Vacation'
      
      python hr.py leave_count 123 20
      
      python hr.py delete employees
      
      python hr.py update leaves 123 '2023-11-22' '2023-11-23' 'Sick leave'
      
      python hr.py remove leaves 123 '2023-11-22'
