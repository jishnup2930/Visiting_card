Title : Generating visiting cards for employees
Author : Jishnu
Date : 14/11/2023

1. OBJECTIVE:
 The init_database function initialize a sql database
 The create_table function create a table in the database
           eg: python gen_vcard.py initdb
 The load_csv_to_database function load a csv file in to table
           eg: python gen_vcard.py load <inputfile>
 The vcard_content function is the format of each vards we creating.
 The qrcode_content function is the format of eacg qr code we creating
 The generate_all_vcards and generate_all_qrcodes functions generates individual vcard and qr code for each employees in the
 provided data

 The gen_sheet.py file generate fake data of employees

2. INPUT:

Csv_file: The path to the CSV file containing contact information.
Each row in the csv_file should contains the  following columns (sample csv file in folder sampledatas)

        firstname, lastname, designation, email, phonenumber


3. OUTPUT:

The code generate one vcard and qr code file for each  row in the data and save the file
in to a subdirectory called 'vcards'.
The file name of each vcard is in the format  
            
            firstnamelastname.vcf

The file name of each qr code is in the format,
            
            firstnamelastname.qr.png          

      * This is a sample vcard file
        
        BEGIN:VCARD
        VERSION:2.1
        N:Bob;John
        FN:John Bob
        ORG:Authors, Inc.
        TITLE:Lawyer
        TEL;WORK;VOICE:666.808.0750x9935
        ADR;WORK:;;100 Flat Grape Dr.;Fresno;CA;95555;United States of America
        EMAIL;PREF;INTERNET:.johnbob@com
        REV:20150922T195243Z
        END:VCARD

        * The qr code contains this data

4. EXECUTION:

The script can be executed from the command line using:
* To print detaiiled logging
          options: 
      -v    Detailed logging
          eg: python gen_vcard.py -v <subcommand>
      -h    Help
            You can use -h after and before of the subcommand to get help

* To create a sql database and tables for employees and leave table
       python gen_vcard.py initdb
      options:
      -v    To print detailed logging   
                  eg: python gen_vcard.py -v initdb
      
      -u   User name of the database
                  eg:python gen_vcard.py initdb -u <your_username>
      
      -d   Database name 
                  eg : python gen_vcard.py initdb -d <database_name>

      -t   Table name 
                  eg: python gen_vcard.py initdb -t <table_name>
      
      -h    Help


* To load a csv file into table
      python gen_vcard.py load <csv_file>
      options:
      -v    To print detailed logging   
                  eg: python gen_vcard.py -v load <csv_file>
      
      -u   User name of the database
      
      -d   Database name 

      -t   Table name 
      
      -h    Help

* To add leave entry to the table
      python gen_vcard.py leave <employee_id> <date> <reason>

* To get the leave leaves_remains
      python gen_vcard leaves_remains <employee_id>

*To generate_all_vcards
      python gen_vcard.py vcard

      options:
      -n'   Number of vcards to generate
                  eg :python gen_vcard.py vcard -n10

      -q    To generate qr code and vcard
                  eg : python gen_vcard.py vcard -q

      -s    To choose the size of the qr code
                  eg : python gen_vcard.py vcard -s200

      -a    To give address in vcard
                  eg:python gen_vcard.py vcard -a 'United states of america'

* To print leave table
      python gen_vcard.py <leave_table>
