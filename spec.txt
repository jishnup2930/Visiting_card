Title : Generating visiting cards for employees
Author : Jishnu
Date : 14/11/2023

1. OBJECTIVE:
 The input_data function reads the provided csv file.
 The vcard_content function is the format of each vards we creating.
 The generate_vcard and generate_qrcode functions generates individual vcard and qr code for each employees in the
 provided csv file.

2. INPUT:

Command-Line Arguments: csv_file: The path to the CSV file containing contact information.
Each row in the csv_file should contains the  following columns (sample csv file in folder sampledatas)

        firstname, lastname, designation, email, phonenumber


3. OUTPUT:

The code generate one vCard and QRCODE file for each  row in the csv_file and save the file
in to a subdirectory called 'vcards'.
The file name of each vcard is in the format  
            
            firstnamelastname.vcf

The file name of each qr code is in the format,
            
            firstnamelastname.qr.png          

      * This is a sample vcard file
        
        BEGIN:VCARD
        VERSION:2.1
        N:Bob;John
        FN:John Bob
        ORG:Authors, Inc.
        TITLE:Lawyer
        TEL;WORK;VOICE:666.808.0750x9935
        ADR;WORK:;;100 Flat Grape Dr.;Fresno;CA;95555;United States of America
        EMAIL;PREF;INTERNET:.johnbob@com
        REV:20150922T195243Z
        END:VCARD

        * The qr code contains this data

4. EXECUTION:

The script can be executed from the command line using:
 
    python3 gen_vcard.py <csvfilename.csv> or python gen_vcard.py <csvfilename.csv>

    * For getting help menu write 
       
          python3 gen_vcard <csvfilename.csv> -h
    
    example:  
       Generates sample employee database as csv

       positional arguments:
        ipfile                Name of input csv file


       options:
      -h, --help            show this help message and exit
      -v, --verbose         Print detailed logging
      -n   NUMBER           Number of vcard to generate [default set to 10]
      -q --qrcode           To generate QR code and vcard
      -s --size             To customise the size of the QR code [default set to 500]
      -a --address          To give address for the vcard [default address is *america]

Example: 
        *python gen_vcard.py <csvfile> -v
         Print detaiiled logging with vcard of default number of employees

        * python genvcard.py <csvfile> -vn5
         To generate 5 vcard with detailed logging
             
        * python genvcard.py <csvfile> -q
        To generate QR code and vacrd for default number of employees

        * python genvcard.py <csvfile> -v -vc
        To generate only vcard for default number of employees
    
        * python gen_vcard.py <csvfile> -qs200
        To generate vacrd and qr code, with qr code size 200x200

        * python gen_vcard.py <csvfile> -a calicut
        To generate vcard with address calicut
